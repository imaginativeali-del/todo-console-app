# Todo Console Application (Phase 1)

## Overview
This project is a simple Todo Console Application built using Python.
It demonstrates basic task management using a terminal-based interface.

## Features
- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks

## Technology Used
- Python 3
- Terminal (Command Line Interface)
