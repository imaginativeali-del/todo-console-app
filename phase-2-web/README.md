# Phase 2: Web-Based Todo Application

## Overview
This phase converts the console-based Todo application into a web-based system.

## Implementation
- Backend implemented using Python FastAPI
- REST API for managing tasks
- In-memory storage for tasks

## Available Endpoints
- POST /tasks → Add a new task
- GET /tasks → Retrieve all tasks

## Status
Basic backend functionality implemented.
Frontend and advanced features can be extended further.
